## 使用方式

1. `下载 Zip压缩包`：[点击下载](https://gitee.com/miyagi-jiye/net-ease-cloud-music/repository/archive/master.zip) or 克隆项目到本地

2. 解压到本地

3. 进入项目目录，安装依赖

   ```sh
   npm install
   ```

4. 运行项目

   ```sh
   npm run dev
   ```

5. 浏览器访问

   ```sh
   http://localhost:5173/
   ```

6. 打包项目
   ```sh
   npm run build
   ```
